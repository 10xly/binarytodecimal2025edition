# binarytodecimal2025edition

Converts from binary to decimal.

```js
const binary2decimal = require('binarytodecimal2025edition')

console.log(btd(11010110)) // 214
console.log(btd("1010")) // 10
console.log(btd(10000000000000000)) // 65536
console.log(btd("7234")) // error because that's not binary
```
